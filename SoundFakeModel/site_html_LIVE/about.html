Project Overview

Project SoundScape is an AI-driven tool for analyzing audio recordings to determine the likelihood of AI-generated content. Project SoundScape utilizes multiple models made for audio incositency detection to determine whether or not a given file is made using generative content or is an authentic clip, this is done by analyzing the results of these seperate models and weighing them together to determine likelihood of AI use.
